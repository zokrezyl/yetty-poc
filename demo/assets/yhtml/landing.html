yetty

A GPU-accelerated terminal emulator with rich content cards

Features

GPU Rendering

WebGPU-based rendering pipeline with SDF primitives, MSDF text, and shader effects. Everything runs on the GPU.

Rich Cards

Embed plots, PDFs, images, HTML, markdown, 3D scenes, and custom shaders directly in your terminal.

HTML/CSS Support

Render HTML with full CSS 2.1 layout via litehtml. Styled content is drawn as GPU-accelerated SDF shapes.

Vector Text

Multi-channel Signed Distance Field fonts stay crisp at any zoom level. No bitmap rasterization needed.

Try it: yetty-client create yhtml -i page.html
